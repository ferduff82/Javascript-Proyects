
Verificar accesos al repo:

ssh git@cmegit


Guardar pass para que no lo pida dos veces:

git config credentials.helper store


Crear agregar y verificar ssh:

https://help.github.com/articles/generating-an-ssh-key/


Desabilitar ssl

git config --global http.sslVerify false


clonar repositorios:

git clone git@cmegit:/marketing_cmegroup_cq5_css.git

git fetch

git pull

Mergear branches:

git checkout master (por ejemplo)
git merge hotfix 
(o sino) 
git merge --no-ff dmt-xxx


Crear un nuevo branch:

git branch -b magicTable


Recuperar archivos borrados:

git checkout .


Ver todas las branches disponibles:

git branch -at


Cambiar nombre a un branch:

git branch -m 'branch-viejo' 'branch-nuevo'


Volver atras despues de un intento de merge fallido

git reset --hard HEAD


COmo usar Stash:

git stash save "my_stash"
